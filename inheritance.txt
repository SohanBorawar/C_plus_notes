Inheritance:-

1) Single-level Inheritance:-

 class A(papa)
     |
 class B(son)

2) Multilevel Inheritance:-

  class A(dada)
     |
  class B(papa)
     |
  class C(beta)


3) Multiple Inheritance:-

 class A     class B
          |
       class C

4) Hierarchical Inheritance:-

  class A 
     |
  class B     class C

5) Hybrid Inheritance:-

---> It is a combination of different Inheritance.
